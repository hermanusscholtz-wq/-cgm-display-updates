Leah CGM Display Firmware v0.10.10

Changes:
- Changes the trend screen from 2 hours to 4 hours.
- Requests up to 64 Nightscout entries and filters the plotted data to the last 4 hours.
- Increases the trend payload read limit to 30000 bytes.
- Optimises the top row so the Wi-Fi bars do not overlap the trend text.
- Replaces the long "Now:" label with only the current glucose value.
- Keeps active LOW/HIGH alarm-limit lines on the graph.
- Keeps local time-in-range calculation from Nightscout entries.
- Keeps night dimming, direct web screen links, update envelope, diagnostics, local setup, security direction, and OTA direction.

Important:
The time-in-range value is calculated locally from the Nightscout entries received by the device.
This display is a secondary CGM awareness and alert companion.
It is not intended to replace the primary CGM app, receiver, or approved medical device.
