Leah CGM Display Firmware v0.10.2

Changes:
- Replaces the old alarm-limits OLED screen with a diagnostics screen.
- OLED now has three screens:
  1. Main glucose screen
  2. 2-hour trend screen
  3. Diagnostics / unit info screen
- Diagnostics screen shows firmware version, build, hardware revision, product model, IP address, Wi-Fi RSSI, Nightscout status, and heap memory.
- Keeps local webserver setup, OTA direction, password/security direction, name layout improvement, and OLED dimming direction.

Important:
This display is a secondary CGM awareness and alert companion.
It is not intended to replace the primary CGM app, receiver, or approved medical device.
